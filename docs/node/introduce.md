# 产品介绍

## 一、产品简介
节点管理是基于 BlueKing Lite 构建的**节点与采集组件管理中枢**，致力于云区域下节点、控制器、采集器与采集配置的集中管控。通过统一完成节点接入、组件安装、配置下发与运行维护，为监控采集、日志收集、CMDB 同步等业务场景提供基础支撑。
凭借轻量化架构、标准化配置体系与安装包治理能力，节点管理模块实现了多云区域、异构操作系统与多 CPU 架构环境下的统一管理，大幅简化云资源与采集组件的全生命周期运维流程。

## 二、产品核心优势
1. **云节点集中管控，打破资源壁垒**
   统一管理多区域网络下的节点资源，可视化展示各区域节点分布与状态，消除分散管理的壁垒，精准掌握全局节点资产。 

2. **自动化运维提效，降低手动成本**
  依托 Linux/Windows 控制器，实现节点上采集器、组件的部署、配置、维护全流程自动化。结合主配置与子配置分离的标准化下发体系，核心操作无需繁琐人工干预，显著提升规模化运维效率。
   
3. **探针与组件生态完整，适配多场景**
  内置开放式的组件库与安装包管理体系，提供如 Telegraf、Beats 等丰富的采集器与 Agent 版本包，兼容 Linux、Windows 及多 CPU 架构环境，全面满足监控、日志、CMDB 等多样化业务下发需求。

## 三、产品应用场景
- **多云区域资源统一管理**
  集中管控不同云区域节点资源，精准掌握各区域节点分布、状态与关联组件，实现云资源全局可视与统一运维。

- **探针全生命周期管理**
  加速节点各类采集器与组件的部署、配置与维护，为监控采集、日志收集、指标上报等业务场景奠定基础。

- **跨节点运维自动化**
  借助控制器实现多节点采集器、组件的批量部署、升级与管控，满足大规模节点集群运维需求，提升规模化运维效率。

- **异构环境适配支撑**
  全面兼容 Linux、Windows 等多系统节点，适配多 CPU 架构资源管理与组件部署，灵活支撑异构环境业务落地。
