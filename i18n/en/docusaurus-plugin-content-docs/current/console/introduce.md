# Product Introduction

## Product Overview
The Console is the first hub-type interface that users see after logging into the BK-Lite platform every day. It serves as the unified entry point for the entire platform, a global message sending and receiving center, and a home base for personal settings. The Console is not merely a collection of menus — it is designed as a highly customizable and cross-device "personal workbench," dedicated to providing IT and business personnel in different roles with the shortest path to information access.

## Core Advantages
*   **Centralized Portal with Unified Authentication**: As an entry bridge spanning all underlying subsystems, it seamlessly integrates with the underlying permission engine to display only the applications that the user has access to after login, avoiding interference from irrelevant business views and preventing unauthorized access.
*   **Personalized Portal Cards**: Different department employees focus on vastly different core systems (e.g., developers focus on AI models and deployments, while operations staff focus on monitoring and alerting). The Console provides a flexible visual drag-and-drop mechanism, allowing each employee to independently subscribe to, hide, and reorder homepage module cards based on their high-frequency business workflows.
*   **Panoramic Cross-module Message Bus**: Aggregates all business notifications delivered from alerting modules, system permission changes, model training completion, and more. Equipped with one-stop batch management and tracking capabilities, it eliminates the frustration of being overwhelmed by scattered internal messages.
*   **Consistent User Management Across Devices**: A unified management point for personal profiles and security options (email rebinding, language preferences, password rotation, etc.). It supports synchronized API endpoints across both mobile APP and web browser, ensuring employee information is accurate around the clock and takes effect system-wide with a single action.

## Use Cases
*   **Agile Business Routing**: When new employees onboard or when organizations deploy new systems, first-time users can complete their default configuration directly through the Console. In daily work, the portal serves as a high-speed switching hub between different sub-applications such as CMDB and Monitoring.
*   **Global To-do Management and Traceability**: When handling alert approvals or system permission audits, users start from the Console's notification bell to review important items requiring attention and complete batch "acknowledged" close-loop operations.
