# Product Introduction

## Overview
OpsPilot is a one-stop enterprise-grade intelligent application building platform developed by BK-Lite. It is dedicated to providing enterprises with end-to-end closed-loop capabilities spanning from underlying model integration, private domain knowledge management, and agent orchestration, to multi-channel conversational delivery. Through a highly configurable workbench and visual flow-based orchestration, enterprises can quickly build AI assistants tailored to their specific business needs.

## Core Advantages
*   **Zero-Code Rapid Development**: Through an intuitive visual drag-and-drop engine and ChatFlow canvas, you can complete agent logic orchestration and publishing without diving into code details.
*   **Deep Integration of Private Domain Knowledge**: Built-in flexible and powerful knowledge base capabilities support various document archiving, Q&A pair extraction, and knowledge graph construction, enabling AI to clearly understand enterprise-specific business context.
*   **Full-Scenario Multi-Channel Delivery**: Build once, deploy everywhere. Built-in support for mainstream channels including WeCom, DingTalk, Web, and mobile apps, breaking down communication barriers.
*   **LLM Ecosystem Compatibility**: Unified management of underlying LLM, Embedding, Rerank, OCR, and other models, free from vendor lock-in, allowing flexible scheduling of different models based on business requirements.
*   **Enterprise-Grade Security & Operations**: Features rigorous permission group management and sensitive configuration encryption mechanisms, along with comprehensive session auditing and token usage analytics, ensuring a secure and controllable integration process.

## Use Cases
*   **Intelligent R&D Assistant**: Connects to internal Git repositories or development platforms, leveraging knowledge graphs to automatically analyze code structures or exception logs, helping developers quickly identify and troubleshoot issues.
*   **Intelligent IT Customer Service**: Extracts enterprise IT service specifications and common Q&As to form a knowledge base, providing 24/7 automated response services through WeCom, reducing repetitive workload for IT support staff.
*   **Data Query & Workflow Agent**: Leverages the OpsPilot toolset to connect to internal enterprise APIs, combined with ChatFlow decision-making capabilities, to help business personnel query metrics or process internal tickets through conversational interactions.
