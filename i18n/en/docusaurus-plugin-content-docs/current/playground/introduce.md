# Product Introduction

## Overview
The Console is the first hub-style interface users see each day after logging into the BK-Lite platform. It serves as the unified entry point for the entire platform, a global messaging center, and a personal settings headquarters. The Console is more than just a collection of menus — it is designed as a highly customizable, cross-device "personal workspace," dedicated to providing IT and business personnel in different roles with the shortest path to the information they need.

## Core Advantages
*   **Centralized Portal with Unified Authentication**: As a gateway bridging all underlying subsystems, it seamlessly integrates with the underlying permission engine to ensure that after login, users only see the applications they are authorized to access, eliminating business perspective interference and unauthorized access.
*   **Personalized Portal Cards**: Different department employees focus on entirely different core systems (e.g., developers focus on large models and deployments, while operations staff focus on monitoring and alerting). The Console provides a flexible visual drag-and-drop mechanism, allowing each employee to subscribe to, hide, and reorder homepage module cards based on their most frequently used business workflows.
*   **Panoramic Cross-Module Message Bus**: Aggregates all business notifications delivered from alerting modules, system permission changes, and even model training completions. It provides one-stop batch management and tracking capabilities, freeing users from the hassle of being buried in scattered internal emails.
*   **Cloud-Consistent User Management**: A unified management point for personal profiles and security options (email rebinding, language preferences, password rotation, etc.). It supports dual-endpoint API synchronization across both the mobile APP and web browser, ensuring employee information is accurate around the clock and takes effect system-wide with a single action.

## Use Cases
*   **Agile Business Routing**: When new employees onboard the platform or when the organization deploys new systems, first-time users can complete default attribute configuration directly through the Console. In daily work, this portal enables high-speed switching between different sub-applications such as CMDB and Monitoring.
*   **Global To-Do Management and Traceability**: When handling tasks such as alert approvals or system permission audits, users start from the Console's notification bell to review important items requiring their attention and complete batch "reviewed" closed-loop operations.
